# C - Hello, World

## Description
This project is an introduction to the C programming language and the compilation process using `gcc`. It covers the four stages of compilation (preprocessing, compilation, assembly, and linking) and basic output functions.

## Learning Objectives
* Why C programming is awesome
* The basics of the `gcc` compiler
* The difference between `printf`, `puts`, and `putchar`
* How to use the `sizeof` operator to check data type sizes
* Following the Betty coding style

## Files
| File | Description |
| --- | --- |
| 0-preprocessor | Script that runs a C file through the preprocessor |
| 1-compiler | Script that compiles a C file without linking |
| 2-assembler | Script that generates assembly code from a C file |
| 3-name | Script that compiles a C file into an executable named `cisfun` |
| 4-puts.c | C program that prints a string using `puts` |
| 5-printf.c | C program that prints a string using `printf` |
| 6-size.c | C program that prints the size of various types |
